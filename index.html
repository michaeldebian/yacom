                    Yacom (Yet Another Container Orchestrator Manager)

The purpose of YACOM is to provide a web based easy to build docker images online. The vision is to allow a single point of management for DEVOPS/Software Engineers which will allow them to add/remove Docker instances on the fly, allowing developers to focus 100% on the code, also, it will add/remove instances to/from load balancers, in this case we will use dedicated instances running nginx.

                             YACOM Architecture
  • Proxy server (nginx)
  • Rsync image repo with docker nodes
  • Easy to build images on demand
  • Reporting capabilities for management per charge code (department)
  • Reporting for admins for resource utilization (pool management)

  • Cassandra based
